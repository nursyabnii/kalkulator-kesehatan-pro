Kalkulator Kesehatan Pro adalah aplikasi berbasis web modern dan responsif untuk menghitung **Indeks Massa Tubuh (IMT/BMI)** serta menganalisis berat badan ideal berdasarkan standar kesehatan. Aplikasi ini menyesuaikan perhitungan secara dinamis berdasarkan parameter **Gender (Pria/Wanita)**, **Usia**, **Tinggi**, dan **Berat Badan**.

## âœ¨ Fitur Unggulan

* **Analisis Gender-Specific**: Perhitungan berat ideal menggunakan rumus **Broca** yang membedakan target antara pria dan wanita.
* **Grafik Gauge Interaktif**: Visualisasi posisi berat badan menggunakan *Gauge Chart* dengan indikator jarum yang dinamis.
* **Update Real-time**: Hasil analisis langsung berubah secara instan saat Anda mengetik tanpa perlu memuat ulang halaman.
* **Kategori Detail**: Klasifikasi berat badan lengkap (8 Kategori) mulai dari "Bobot Terlalu Rendah" hingga "Obesitas Kelas III".
* **Desain Modern & Responsif**: Tampilan UI bersih bergaya aplikasi mobile, nyaman digunakan di smartphone maupun desktop.
* **Indikator Selisih**: Memberikan informasi tepat berapa kilogram kelebihan atau kekurangan berat badan untuk mencapai angka ideal.
